#  Smart Energy Data Visualizer using Python & IoT

This project visualizes real-time and offline energy meter data using Streamlit and MQTT.  
It allows anomaly detection, trend analysis, and interactive visualization for efficient energy management.

## Features
- Real-time IoT data visualization via MQTT
- CSV data upload and analysis
- Anomaly detection using statistical methods
- Interactive Streamlit UI with Matplotlib and Plotly graphs

## Tech Stack
- Python (Pandas, Matplotlib, Streamlit, Paho-MQTT)
- IoT Sensor Data
- MQTT Protocol

## How to Run
```bash
pip install -r requirements.txt
streamlit run smart_energy_visualizer_ui.py
